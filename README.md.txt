A comprehensive machine learning dashboard for e-commerce customer analytics, featuring association rule mining, classification, clustering, regression, and dynamic pricing.

## üåü Features

### üìä Analytics Modules

1. **Association Rule Mining**
   - Discover product purchase patterns
   - Interactive support, confidence, and lift thresholds
   - Network graph visualization
   - Export rules to CSV

2. **Multi-Class Classification**
   - Predict customer willingness to pay
   - Compare multiple algorithms (Logistic Regression, Random Forest, XGBoost, SVM)
   - Confusion matrix and ROC curves
   - Feature importance analysis

3. **Customer Clustering**
   - Segment customers using K-Means, Hierarchical, or DBSCAN
   - Elbow method and silhouette analysis
   - PCA visualization
   - Detailed cluster profiles

4. **Regression Analysis**
   - Predict numeric willingness to pay
   - Compare 5 regression models
   - Actual vs predicted plots
   - Residual analysis

5. **Dynamic Pricing Engine**
   - Real-time price prediction
   - Customer-specific pricing recommendations
   - Confidence intervals
   - Volume-based pricing strategies
